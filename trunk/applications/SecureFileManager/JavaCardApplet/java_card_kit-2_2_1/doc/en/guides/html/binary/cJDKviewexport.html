Chapter   6

Viewing an Export File

The exp2text tool is provided to allow you to view any export file in text format. 

exp2text  [options] <package_name>

Where options include:

TABLE 10  –  exp2txt Command Line Options
Option
Description
-classdir <input root directory>
Specifies the root directory where the program looks for the export file.
-d <output root directory>
Specifies the root directory for output.
-help
Prints help message.
